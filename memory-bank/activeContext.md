# Active Context

## Current Work
Имплементирана е новата функционалност за URL база данни с JSON файл и автоматично инкрементиране.

## Recent Changes
- Създаден `url_manager.py` - модул за CRUD операции с `urls.json`
- Добавени API endpoints в `app.py`: GET/POST `/api/urls`, `/api/urls/add`, `/api/urls/delete`, `/api/urls/update`
- Променен `scraper.py` - поддръжка на `url_index` параметър и автоматично инкрементиране след сваляне
- Променен `templates/index.html` - нов UI панел "Моите URL адреси" с добавяне, изтриване, редактиране и зареждане на URL

## Next Steps

- JSON структурата е максимално опростена - само масив от URL низове
- UI показва максимум ~10 реда наведнъж със скрол
