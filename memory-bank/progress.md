# Progress

## What Works
- **`url_manager.py`**: Пълен CRUD модул за `urls.json`
  - `load_urls()` / `save_urls()` - зареждане и запис
  - `add_url()` - добавяне с проверка за дублиране по основна директория
  - `delete_url()` - изтриване по индекс
  - `update_url()` - промяна с проверка за дублиране
  - `get_base_url()` - нормализиране на URL до протокол + домейн + 1 ниво
  - `increment_path_number()` - инкрементиране на число в директорията преди файла
  - `increment_url()` - инкрементиране със запис в JSON

- **`app.py`**: Нови API endpoints
  - `GET /api/urls` - връща списъка
  - `POST /api/urls/add` - добавя URL
  - `POST /api/urls/delete` - изтрива URL
  - `POST /api/urls/update` - променя URL
  - `GET /download` - приема `url_index` параметър

- **`scraper.py`**: Поддръжка на URL индекс
  - `download_images()` приема `url_index` параметър
  - След успешно сваляне автоматично инкрементира URL в JSON
  - Изпраща `increment_info` SSE съобщение

- **`templates/index.html`**: Нов UI панел
  - Панел "Моите URL адреси" в дясната колона
  - Поле за добавяне на нов URL (с Enter поддръжка)
  - Скролиращ се списък (макс ~10 реда)
  - Радио бутон за избор на URL
  - Бутони за редактиране (✏️) и изтриване (🗑️)
  - Бутон "📋 Зареди URL" - добавя избрания URL в полето за обработка
  - Бутон "🔄 Презареди" - опреснява списъка
  - Обработка на `increment_info` събитие - показва в лога и обновява списъка
  - Поддръжка на тъмна тема
